## Placeholder Images (ใช้ภายนอกชั่วคราว)

ในโค้ดจะใช้ URL จาก:
- **Unsplash** (รูปจริงคุณภาพสูง)
- **Placeholder services** สำหรับ development

## เมื่อมีรูปจริง

วางรูปในโฟลเดอร์นี้:
- `hero-bg.jpg` - พื้นหลัง hero section
- `categories/` - ไอคอนหมวดสินค้า
- `services/` - รูปบริการ
- `testimonials/` - รูปลูกค้า
